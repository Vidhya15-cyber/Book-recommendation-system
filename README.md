# 📚 Book Recommendation System

## 📌 Overview

The Book Recommendation System is a machine learning application that recommends books based on user preferences. It combines **Content-Based Filtering**, **Collaborative Filtering**, and a **Hybrid Recommendation Approach** to generate personalized recommendations through an interactive Streamlit web application.

---

## ✨ Features

- 📖 Content-Based Book Recommendation
- 👥 User-Based Collaborative Filtering
- 🔀 Hybrid Recommendation System
- 📊 Similarity-based recommendations
- 💻 Interactive Streamlit interface
- ⚡ Fast and personalized book suggestions

---

## 🧠 Recommendation Techniques

### 📖 Content-Based Filtering
Recommends books based on title, author, and genre similarity using **TF-IDF Vectorization** and **Cosine Similarity**.

### 👥 Collaborative Filtering
Recommends books by identifying users with similar reading preferences and suggesting books they highly rated.

### 🔀 Hybrid Recommendation
Combines Content-Based and Collaborative Filtering to improve recommendation accuracy and provide more relevant suggestions.
